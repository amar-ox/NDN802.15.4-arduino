# NDN802.15.4-arduino


## What is it?

*NDN-802.15.4* is an Arduino library for using [Named Data Networking](http://named-data.net/) over IEEE802.15.4.

*Work in progress...*

## It implements
* The three NDN data structures (FIB, PIT, CS)
* An AI-inspired wireless forwarding strategy
* Generic packet representation (TLV enconding support is in progress)
